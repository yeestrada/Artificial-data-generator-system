*******************Acerca de*******************
El sistema generador de datos artificiales constituye un resultado del tema de investigación: "GENERACIÓN DE DATOS ARTIFICIALES EN PROBLEMAS DE CLASIFICACIÓN CON CLASES NO BALANCEADAS", el cual se encarga de la generación de datos para problemas de clases no balanceadas haciendo uso del algorítmo Sobremuestreo de densidad estadística propuesto en la investigación referenciada.

*******************About*******************
The artificial data generator system is a result of the research topic: "GENERATION OF ARTIFICIAL DATA IN CLASSIFICATION PROBLEMS WITH UNBALANCED CLASSES", which is in charge of generating data for problems of unbalanced classes making use of the algorithm Oversampling of statistical density proposed in the referenced research.

*******************Installation guide*******************
It is important to mention that this installation will install all the necessary dependencies and create a new folder on your disk with the necessary files to deploy the proposed system.

Steps to install on windows operating system:
 You must run the install.bat file with administrator rights.
 
 At the end of the process, an SDO.bat file will remain on your desktop.

*******************Guia de Instalación*******************
Es importante mencionar que esta instalación instalará todas las dependencias necesarias y creará una carpeta nueva en su disco con los archivos necesarios para desplegar el sistema propuesto.

Pasos para instalar en sistema operativo Windows:
 Debe ejecutar el fichero install.bat con permisos de administrador.

Al finalizar el proceso, quedará en su escritorio un fichero SDO.bat.
